This program is used to find lines, words inside any readable files in specified directory recursively

In the field "Keyword or line"
Insert the keyword or line youre searching for

In the field file name regex
Insert the file name or its regex(file name or a regex if you roughly remember which file contains the line)

The two checkboxes at the bottom of the window are used to make the search 
case-insensitive or to use the keyword as a regex

NOTE!!!
PROBLEMS CAN OCCURE IF YOU TRY TO FIND KEYWORD IN A PATH THAT NEEDS ADMINISTRATIVE PREVELEGES. 
RUNNING CODE AS ADMIN MAY OR MAY NOT SOLVE IT
